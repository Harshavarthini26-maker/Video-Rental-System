# Video Rental System 🎬

## Project Overview

The **Video Rental System** is a Java-based console application that allows users to manage video rentals. The system provides basic operations like adding videos, checking availability, renting videos, and returning videos.

This project demonstrates the use of **Object-Oriented Programming (OOP) concepts in Java** such as classes, objects, constructors, methods, and ArrayList collections.

## Features

* Add new videos to the store
* Display available videos
* Rent a video
* Return a rented video
* Track video availability
* Menu-driven console interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Java Collections Framework (ArrayList)

## Class Description

### Video.java

* Represents a video object.
* Stores video details such as title and availability status.
* Contains methods to rent and return videos.

### VideoStore.java

* Manages the collection of videos.
* Handles adding, renting, returning, and displaying videos.

### VideoLauncher.java

* Main class used to run the application.
* Provides a menu-based user interface.

## How to Run the Project

1. Clone the repository:

```
git clone <repository-url>
```

2. Open the project folder.

3. Compile the Java files:

```
javac Video.java
javac VideoStore.java
javac VideoLauncher.java
```

4. Run the application:

```
java VideoLauncher
```
